# Icons Directory

This directory is for SVG icons used in the calculator interface.

## Icon Usage

Icons can be used for:
- Scientific mode function buttons
- Menu items
- Theme switcher
- Mode indicators

## Recommended

- **Feather Icons** - Clean, minimal SVG icons
- **Material Icons** - Google's icon set
- **Font Awesome** - Comprehensive icon library

## Current Status

The calculator currently uses text-based buttons. Adding icons here will enhance the visual appeal.
