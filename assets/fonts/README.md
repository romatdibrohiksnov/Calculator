# Fonts Directory

This directory is for custom digital/mono fonts to enhance the calculator display.

## Recommended Fonts

You can download and add fonts like:
- **Orbitron** - Modern geometric font
- **Segment7** - Classic digital calculator display
- **Roboto Mono** - Clean monospace font
- **Source Code Pro** - Developer-friendly monospace

## Installation

1. Download .ttf or .otf font files
2. Place them in this directory
3. The application will automatically load them on startup

## Current Fallback

The calculator currently uses system monospace fonts:
- Windows: Consolas
- macOS: Monaco
- Linux: Liberation Mono
